CLI

Command-line interface for the Lattice network. Manage nodes, query chains, send transactions, and generate Merkle proofs from your terminal.

Install

# Build from source
git clone https://github.com/treehauslabs/lattice-cli.git
cd lattice-cli
swift build -c release

# Move to PATH
cp .build/release/lattice /usr/local/bin/

# Verify
lattice --version

Configuration

The CLI reads configuration from ~/.lattice/config.toml. Set it up once and all commands use the same node connection. 

# Initialize default config
lattice init

# Point to a running gateway
lattice config set --gateway http://localhost:8545

# Set default chain
lattice config set --chain payments

# View current config
lattice config show

Commands

Node

lattice node status
Show connected node's sync state, peer count, and chain heights
lattice node peers
List connected peers with reputation scores and latency
lattice node chains
List all chains in the hierarchy with heights and hash rates

Blocks

lattice block get <height|hash>
Fetch a block by height or hash. Displays header, tx count, and state root.
lattice block latest
Show the latest block on the current chain
lattice block watch
Stream new blocks as they are committed (Ctrl+C to stop)

Transactions

lattice tx get <hash>
Fetch transaction details and confirmation status
lattice tx send --to <addr> --amount <n>
Sign and submit a transaction. Uses the default wallet key.
lattice tx pending
List pending transactions in the mempool

Cross-Chain

lattice transfer --from <chain> --to <chain> --amount <n>
Initiate a cross-chain transfer with automatic Merkle proof generation
lattice proof get --chain <id> --key <key>
Generate a Merkle inclusion proof for a state key
lattice proof verify <proof-file>
Verify a Merkle proof against the current chain state

State

lattice state get <key>
Read a value from the current chain's state tree
lattice state list --prefix <prefix>
List state keys matching a prefix

Wallet

lattice wallet create
Generate a new keypair and save to the local keystore
lattice wallet list
List all keys in the local keystore
lattice wallet balance [--chain <id>]
Check balance for the active key, optionally on a specific chain

Examples

Check node status and latest block

$ lattice node status
Node:     127.0.0.1:8545
Synced:   true
Height:   184,293
Peers:    12
Chains:   7

$ lattice block latest
Height:     184,293
Hash:       a1b2c3d4e5f6...
State Root: 789abcdef012...
Txns:       23
Time:       2.4s ago

Send a cross-chain transfer

$ lattice transfer --from payments --to identity --amount 100
Deposit tx:   0xabc123... (payments, height 184,294)
Proof:        generated (3 nodes, 96 bytes)
Withdrawal:   0xdef456... (identity, pending)
Status:       confirmed in 2 blocks

Shell Completions

# Bash
lattice completions bash > /etc/bash_completion.d/lattice

# Zsh
lattice completions zsh > ~/.zsh/completions/_lattice

# Fish
lattice completions fish > ~/.config/fish/completions/lattice.fish
